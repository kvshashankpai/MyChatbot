## Step 0 вЂ” Manual Dataset Downloads (required before anything else)

### EmpatheticDialogues
- URL: https://www.kaggle.com/datasets/atharvjairath/empathetic-dialogues-facebook-ai
- Download the zip and extract. The dataset is a **single combined CSV file** named `emotion-emotion_69k.csv` (~69k rows, all splits merged).
- Place it at:
  ```
  data/raw/empathetic_dialogues/emotion-emotion_69k.csv
  ```
- CSV columns used by the code: `conv_id`, `utterance_idx`, `context`, `utterance`

### ESConv
- URL: https://github.com/thu-coai/Emotional-Support-Conversation
- Clone the repo or just download `data/ESConv.json` directly from it.
- Place the file at:
  ```
  data/raw/ESConv.json
  ```
- JSON structure: top-level list of dialogue objects, each with keys
  `emotion_type`, `problem_type`, `situation`, `emotion_intensity`, `dialog`

## Step 2b вЂ” Prepare ESConv Data (required before Phase 2)

The raw `ESConv.json` schema differs from what `dataset.py` expects (different
role names, strategy field location, missing intensity field, and inconsistent
strategy label strings). `prepare_esconv.py` normalises the raw file and calls
`dataset.extract_esconv_qa_pairs()` internally to produce correctly formatted
JSONL files with the sliding-window context, metadata tokens, and integer
`emotion_label` / `strategy_label` fields.

```bash
python prepare_esconv.py
```

With explicit paths:
```bash
python prepare_esconv.py \
  --input    data/raw/ESConv.json \
  --out_dir  data/processed \
  --val_frac  0.15 \
  --test_frac 0.15 \
  --seed      42
```

This produces:
- `data/processed/esconv_train.jsonl` вЂ” ~70% of conversations (~1,850 pairs)
- `data/processed/esconv_val.jsonl`   вЂ” ~15% of conversations (~450 pairs)
- `data/processed/esconv_test.jsonl`  вЂ” ~15% of conversations (~370 pairs)
- `data/processed/strategy_counts.json` вЂ” strategy frequencies from train split only
  (used for class-weighted loss in Phase 2)

**What the normalisation fixes:**
- Role names: `"speaker"` в†’ `"seeker"`, `"listener"` в†’ `"supporter"`
- Strategy field: moved from `turn["annotation"]["strategy"]` в†’ `turn["strategy"]`
- 12 raw strategy variants mapped to the 8 canonical labels (`ESCONV_STRATEGY_MAP`)
- Missing `emotion_intensity` field defaulted to `3`
- Emotions not in `ESCONV_EMOTION_TO_ID` (`depression`, `shame`, `nervousness`)
  are mapped to `neutral` (id=7)

## Architecture Summary

| Component        | Spec value                                      |
|-----------------|-------------------------------------------------|
| Encoder          | 4 layers, 2-head self-attention, d_model=256    |
| Decoder          | 4 layers, masked self-attn + cross-attention    |
| d_k per head     | 128 (= 256 / 2 heads)                           |
| d_ff             | 1024 (4Г— expansion)                             |
| Vocabulary       | ~10,000 BPE tokens (trained on project corpus)  |
| Emotion head     | 2-layer MLP on [CLS] в†’ 32 classes (Phase 1)     |
| Strategy head    | 2-layer MLP on [CLS] в†’ 8 classes                |
| Weight tying     | embedding.weight == output_proj.weight          |
| Positional enc.  | Fixed sinusoidal (pre-computed buffer), max=512 |
| Inference        | Nucleus (T=0.85, p=0.9) / Beam search (w=4)     |

## Loss Function
```
L_total = L_gen + 0.3 * L_emotion + 0.3 * L_strategy
```
- `L_gen`:      Label-smoothed cross-entropy (smoothing=0.1, ignore_index=-100)
- `L_emotion`:  CrossEntropyLoss on [CLS] emotion head (ignore_index=-1)
- `L_strategy`: Class-weighted CrossEntropyLoss on strategy head (ignore_index=-1)
  - Phase 1: lambda_strategy=0.0 (masked вЂ” no strategy labels in EmpatheticDialogues)
  - Phase 2: lambda_strategy=0.3 (active, class-weighted for ESConv imbalance)

## Evaluation Targets

| Metric       | Target  | Why                                           |
|-------------|---------|-----------------------------------------------|
| Perplexity  | < 80    | Syntactic fluency                             |
| Distinct-1  | > 0.15  | Word-level response diversity                 |
| Distinct-2  | > 0.40  | Bigram diversity вЂ” primary early-stop metric  |
| Emotion F1  | > 0.65  | Weighted F1 over 8 emotion classes            |
| Strategy F1 | > 0.60  | Weighted F1 over 8 strategy classes           |
